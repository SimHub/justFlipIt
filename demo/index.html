OPTION

Attribute Possible value Default Description

Click

false && true && '[elementName]'

false

 You can specify a element for the clickEventHandler
$('[elementName]').justFlipIt({ Click:'[elementName]' });

FlipX

true

false

 The axis that you want to rotate around.
$('[elementName]').justFlipIt({
FlipX:true
});
Will rotate on the X axes.

Template

' [html or plain text] '

 You can specify the back site
$('.flip-04').justFlipIt({
Template: '< h1>HELLO< /h1>'
});

Style

Style:[
{
el:'[elementName]',
style:{'property':'value'}
},
...
]

[ ]

 You can add custom css styles :)

$('[elementName]').justFlipIt({
Style:[
{el:'[elementName1]',style:{'property':'value'}},
{el:'[elementName2]',style:{'property':'value'}},
...
];

if you need to style the plugins root element use 'self'

$('[elementName]').justFlipIt({
Style:[
{
el:'self',
style:{'property':'value'}
},
...
];
