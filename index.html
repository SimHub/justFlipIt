Options

[ Click, FlipY, Template ]

Attribute Possible value Default Description
Click false , true && '[elem:name]' false You can specify a element for the clickEventHandler
$('[element:name]').justFlipIt({ Click:'[element:name]' });

CAUTION:
If you also want to change the content on the back,
you have to name the target element inside your Template:[property***]
the same as in your Class:[property***]

$([elem:name]).justFlipIt({
Click: '[elem:name***]',
Template: '
< a style="cursor:alias" class="[elem:name***]">
< img class="ui avatar image " src="img/stevie.jpg" alt="avatar">
< em>< strong style="color: #a4363b;">Now Click here< /strong>< /em>< /a>
'});
FlipY true false = X The axis that you want to rotate around.
$('[elem:name]').justFlipIt({ FlypY:true });
Will rotate on the Y axes.
