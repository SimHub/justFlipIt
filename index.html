Options

[ Click, FlipX, Template ]

Attribute Possible value Default Description
Click false , true && '[element:name]' false You can specify a element for the clickEventHandler
$('[element:name]').justFlipIt({ Click:'[element:name]' });

CAUTION:
If you also want to change the content on the back,
you have to name the target element inside your Template:[property***]
the same as in your Class:[property***]

$([element:name]).justFlipIt({
Click: '[element:name***]',
Template: '
< a style="cursor:alias" class="[element:name***]">
< img class="ui avatar image " src="img/stevie.jpg" alt="r">
< em>< strong style="color: #a4363b;">Now Click here< /strong>< /em>< /a>
'});
FlipX true false = Y The axis that you want to rotate around.
$('[element:name]').justFlipIt({ FlipX:true });
Will rotate on the X axes.
